# 同步
```
定义：同步是指任务按照顺序执行，一个任务完成之后，才开始下一个任务，在这种模式下，每个任务都会阻塞后续任务的执行，直到自己完成
同步代码按照顺序执行，会阻塞后续代码的执行
```

``` javascript
function syncTask() {
    console.log("任务1开始");
    console.log("任务1结束");
    
    console.log("任务2开始");
    console.log("任务2结束");
}

// 输出顺序：任务1开始 -> 任务1结束 -> 任务2开始 -> 任务2结束
```

# 异步
```
定义：异步则允许任务在不阻塞后续任务的情况下执行。任务可以被分派出去，不必等待其完成就继续执行后续任务，异步任务会在其完成时通过回调、事件、Promise等机制通知结果
```

``` javascript
function asyncTask() {
    console.log("任务1开始");
    
    setTimeout(() => {
        console.log("任务1结束"); // 异步完成
    }, 1000);
    
    console.log("任务2开始");
    console.log("任务2结束");
}

// 输出顺序：任务1开始 -> 任务2开始 -> 任务2结束 -> 任务1结束
```
